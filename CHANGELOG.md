## [0.1.0] - 2024-01-01

### Added

- Initial release
- Markdown slide parsing with `---` delimiters
- YAML frontmatter support (title, author)
- Auto-generated title slide from frontmatter
- Keyboard navigation (arrows, vim keys, home/end)
- Glamour-powered markdown rendering with syntax highlighting
- Lipgloss-styled UI with adaptive light/dark mode
- Status bar with slide counter
- Fang-powered CLI with styled help output
